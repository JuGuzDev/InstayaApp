# InstayaApp
Esta app tiene seis archivos, Home, Login, Register, Create, Update, View. Que se encuentran en la carpeta src/pages
las cuales corresponden a las paginas.

Home -- tiene un boton que conduce al login.

Login -- activa la cuenta y si no tiene cuenta conduce a la pagina de registro.

Register -- aqui se puede registrar nueva cuenta.

Create -- Se crea la orden del envio.

View -- Muestra una lista de las ordenes hechas y dando clic en numero de orden te conduce a Update.

Update -- Se puede cambiar informcion o eliminar entrega siempre y cuando no este en estado entregada.


### Instalacion

puedes descargarlo como archivo .zip o tambien clonarlo 
`$ git clone url`

cuando lo quieras correr en un editor tienes que instalar modulos con
`$ npm install`
